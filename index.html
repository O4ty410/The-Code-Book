HTML — The Skeleton of Every Page
Every webpage you've ever visited is built with HTML at its core. HTML gives a page its structure. It says: here is a heading. Here is a paragraph. Here is a button.

Think of it like the frame of a house. Before the walls, before the paint — there's a frame. HTML is the frame.

Every HTML element follows the same pattern. An opening tag that names what it is. The content that goes inside it. A closing tag that ends it. That's the entire language in one sentence.
THE PATTERN
<h1>This is a heading</h1>
Open tag → content → close tag. Every element. Every time.
